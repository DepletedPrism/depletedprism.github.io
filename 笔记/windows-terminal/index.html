Windows Terminal 备忘录

2020-03-21
  • 笔记

看到群友的终端好高级啊 (

于是就直接去搜了 “Windows Terminal”… 没想到还真的有这个东西, 也真的叫这个名字 = =

记录以备忘.

相关命令

  • wt

    用于启动 Windows Terminal.

    Win+R 运行此命令就好了.

    在资源管理器中 Win+L 并输入 wt 也可以打开 Windows Terminal.

    不过此时打开的是默认目录, 在配置文件中添加 "startingDirectory": "." 即可打开当前目录.

配置文件

首先有官方文档: https://github.com/microsoft/terminal/blob/master/doc/user-docs/UsingJsonSettings.md

依葫芦画瓢即可. 在此存档我个人配置的部分内容…
