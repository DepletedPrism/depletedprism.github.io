HAOI 2019 游记

2019-04-13
  • 随笔

阽余身而危死兮，览余初其犹未悔。

概况：某高一入坑的 OIer，划水参加十二省联考的咸鱼

Day -1

清明放假。
清明后就期中了，依稀记得 NOIp 时间也和期中考试冲突了
那政史地老师，对不起了 Orz

Day 0

上午十点半起床，然后愉快去刷文化课了，因为太菜了不想去写板子啊
其间颓废不提。

好吧，还是去洛谷看了看，发现日报更新了，于是复习了 ST 表，只是之前学的太粗浅，模板也写的太丑，重写了模板之后写了写日报安利的题，喜闻乐见地写锅了 = =
静态差错发现是预处理时边界的问题，写反了一个符号

考场上 Debug 就真的透心凉了啊…

Day 1

  • 00:03

    某咕打卡大凶，怕不是要凉
    我会告诉你隔壁 @Crafty_L 与 @良辰、 大吉吗 = =

  • 8:00

    遇到的监考老师都好眼熟，怎么和 NOIp2018 的那波人那么像呢…
    开场解压试题还是出了点问题，离机房前面的白板太远了什么都看不清 = =

    看完 T1 随手写了 $O(n^{2}\log n)$ 的枚举左右端点 + STL 优先队列找前 k 大的贪心，跑大样例却没有过，感觉像是 %lld 的锅就没有继续管了

    T1：期望得分 40 pts

    T2 有 AC 自动机的既视感，（但他们说是后缀自动机？），依稀记得寒假前 @maomao9173 在我学 KMP 的时候推荐去顺手学个 AC 自动机，但是我成功地咕咕咕…

    唯一的亮（cao）点（dian）就是这个了：

    十二省联考命题组温馨提醒您：

    数据千万条，清空第一条。
    多测不清空，爆零两行泪。

    T2：期望得分 0 pts

    T3 感受到了出题人深深的恶意（雾

    “就算你知道所有的测试点，你能切掉这道题吗？”

    干瞪眼查出 $1_998244353$ 的规律，只是写了个循环找到模数，然后发现是 $998244353$，怎么这么眼熟呢 = =

    然后看 $1?$，发现输入炸掉了 long long， 开始考虑能不能用 Luogu-P5091 一样的方式写过去，发现还是不可做 = =

    感觉 $1wa_998244353$ 更友善一点，用 int 的自然溢出与 $\mod 998244353$ 过了测试点 #6，#7 还是写不出来啊 = =

    考试结束还有 1h 的时候看了看 $2p$，$2u$，$2g$ 之类，已经无心找规律了

    T3：期望得分 18 pts

  • 12:50

    提交的时候 ftp 出现了小问题，等到 $12:40$ 终于好了，出门看到 star，以及一语不发的学长和延时 15min 的 guyan。Day1 完结，下午讲课不想去听了

  • Day1 期望得分 $40 + 0 + 18 = 58\ \text{pts}$

Day 2

  • 洛谷打卡总算是大吉了

  • 8:00
    这次拿到题目就比较顺利了，把题目翻完一遍就去想 T1，理解完题意之后看到 $n \leq 1$ 的福利数据之后果断特判。接着除了模拟 + 爆搜之外就没有什么思路了。写完 T2 的暴力回头再写这里的模拟，没有过第一个样例的第二个点，想不出哪里出锅了 = =。

    T1：期望得分 0 pts

    于是想了想 T2，考虑完样例之后有树链剖分的风格。可惜我不会这个 = =，只能看看有没有搜索可以利用的性质。最后写了个从叶子往根找，依次枚举叶子权值的暴力。顺利把三个样例过完之后就不管了。

    T2：期望得分 20 pts

    T3 想搞搞 $k = 1 $ 的点，还有半小时，感觉时间有点来不及，于是 BFS 没有打，开始用样例盲猜 $k = 1 $ 且 $L = 1 $ 的结论。结果喜闻乐见的挂了 = =

    T3：期望得分 0 pts

  • Day2 期望得分 $0 + 20 + 0 = 20\ \text{pts}$

  • 下午返校准备期中考试去了，也没有去上课。

  • 考完之后去看昨天的榜，B 类一群爆零 = =，弱省的实力见证时刻。Day 1 的 T1 挂了好多，暴力分也没有了，只拿到了 T3 的 14 pts。

后记

几天后公布了省队的名单以及 Day 2 的代码，去洛谷上交了下，T1 拿到了 10 pts，其余的全部 TLE 了。T2 除了前五个点也是 TLE，拿到了前三个点的分数，很好奇那两个点是怎么 WA 的 = =。

近半年来最后悔的事就是没有抓紧时间学 OI 的知识点吧，去年 NOIp 前后一直在写一些搜索或者是模拟之类的东西，今年寒假也没有去成集训。考完之后真的有“书到用时方恨少”的感受。

最近竞赛又受到一次波动，在升学的问题面前，单纯的热爱依旧弱不禁风。只是希望在退役之前，能够在 OI 上多存在一点时间了。
